1) 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
- выполнить команду "git clone" со скопированным URL репозитория. Предварительно нужно создать аккаунт в GitHub
  и добавить ключ SSH
2) Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
- выполнить команду "git unit" в терминале или командной строке, перейдя в каталог на компьютере
- файл добавить командой "git add"
- отправить "git push"
3) Как получить на локальный компьютер изменения с github.com
- команда "git pull"
4) Какую информацию нам дают команда :
git status  -  показывает текущее состояние рабочего каталога по сравнению с после последнего сделанного коммита
git log  - показывает историю коммитов
git remote -v   -  показывает информацию об удаленных репозиториях, связанных с локальным нашим репозиторием
